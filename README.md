# AD Account Automation

PowerShell scripts built to automate Active Directory user provisioning.

## Overview
This project was created to reduce manual account setup time and improve consistency during user provisioning. It includes both a manual account creation script and a bulk CSV-based provisioning script.

## Included Scripts
- `New-ADUserAccount.ps1`  
  Bulk creates AD users from a CSV file and can assign group membership.

- `ADUserAccountManual.ps1`  
  Creates a single AD user manually and can assign group membership.

## What It Does
- Automates Active Directory user account creation
- Supports manual and bulk provisioning workflows
- Assigns department and title values
- Adds users to groups
- Helps standardize account setup across users

## Technologies Used
- PowerShell
- Active Directory

## Why I Built It
I wanted to improve the speed and consistency of account provisioning while building a more practical automation workflow.
